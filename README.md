# Game-Chat-Bot (Бот-консультант по играм)
# Telegram Bot • PyTelebotAPI • SQLite3
# Версия: 1.0.0 | Статус: Активен

## 📖 О проекте

Telegram-бот, который выступает в роли персонального консультанта по играм. Бот помогает пользователям находить информацию о видеоиграх, рекомендует случайные игры и позволяет составлять коллекцию избранного. Проект демонстрирует работу с базами данных, системой поиска и пользовательскими предпочтениями.

## ✨ Основные возможности

- 🎮 Поиск игр - консультация по названиям с нечетким поиском
- 🎲 Рекомендации - случайные игры для знакомства
- ⭐ Личная коллекция - добавление игр в избранное
- 📋 Список желаемого - просмотр сохраненных игр
- 📊 Детальная информация - год выпуска, платформа, жанр, издатель, продажи

## 🤖 Команды бота

| Команда | Описание | Пример |
|---------|----------|--------|
| `/start` | Начало работы, приветствие | `/start` |
| `/random` | Получить рекомендацию случайной игры | `/random` |
| `/favorites` | Показать список избранных игр | `/favorites` |

## 🎮 Процесс работы

```
1. 👤 Пользователь → /start
   🤖 Бот: Приветствие и инструкции

2. 👤 Пользователь → "Mario"
   🤖 Бот: Поиск и результаты
   │
   ├── Если 1 игра → Детальная информация + кнопка "В избранное"
   └── Если несколько → Список для уточнения

3. 👤 Пользователь → нажимает "В избранное"
   🤖 Бот: Игра сохраняется в личную коллекцию

4. 👤 Пользователь → /favorites
   🤖 Бот: Список сохраненных игр
```

## 🔧 Технические детали

**Инициализация базы данных:**
```python
def init_database():
    """Создает таблицу избранного при первом запуске"""
    cur.execute("""
        CREATE TABLE IF NOT EXISTS favorites (
            user_id INTEGER,
            game_id INTEGER,
            added_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
            PRIMARY KEY (user_id, game_id)
        )
    """)
```

**Поиск игр:**
```python
# Нечеткий поиск по названию (без учета регистра)
cur.execute("SELECT * FROM mytable WHERE LOWER(Name) LIKE ?", 
           ('%' + message.text.lower() + '%',))
```

**Система избранного:**
```python
# Добавление в избранное
cur.execute("INSERT INTO favorites (user_id, game_id) VALUES (?, ?)",
           (user_id, game_id))

# Получение избранного
cur.execute("""
    SELECT g.* FROM mytable g
    JOIN favorites f ON g.Rank = f.game_id
    WHERE f.user_id = ?
""", (user_id,))
```
